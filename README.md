# PostHubAPI

**PostHubAPI** is a blog API providing CRUD functionalities for Posts and Comments.

## Technologies Used

- **Spring Boot**: Framework for building Java-based web applications.
- **JPA (Java Persistence API)**: Java specification for accessing, managing, and persisting data between Java objects and relational databases.
- **H2 Database**: In-memory database for quick development and testing.
- **Spring Validation**: Part of the Spring framework for declarative validation of JavaBeans.

## Features

- **CRUD for Posts**: Create, read, update, and delete blog posts.
- **CRUD for Comments**: Manage comments associated with each post.
